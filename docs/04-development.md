# 04 - Development

## Coding Standards
- Python 3.11+
- Typed functions and clear module boundaries
- Validation at API boundaries using Pydantic

## Implementation Sequence
1. Data model and DB session setup.
2. Station and reading APIs.
3. AQI computation and persistence.
4. Alert policy and alert lifecycle endpoints.
5. Test suite and documentation.

## Branching and Review Strategy
- Main branch protected.
- Feature branches merged with code review.
- Test execution required before merge.
